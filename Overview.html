03 النماذج المستخدمة
📐
OLS — Ordinary Least Squares
بيقلل الـ Residual Sum of Squares (RSS). بيدينا estimates غير متحيزة (unbiased) تحت شروط Gauss-Markov. بنستخدم statsmodels عشان ناخد Standard Errors، t-statistics، و p-values لكل coefficient.
Ŷ = β₀ + β₁X₁ + ... + β₈X₈ + ε
🏔️
Ridge Regression (L2)
بتضيف L2 penalty للـ objective function: بتقلل من الـ variance على حساب bias بسيط. مفيدة لما يكون في multicollinearity. استخدمنا alpha = 1.0
min ||y − Xβ||² + α||β||²
🎯
Lasso Regression (L1)
بتضيف L1 penalty — ممكن تخلي بعض الـ coefficients يوصلوا لـ صفر بالظبط (automatic feature selection). استخدمنا alpha = 0.1
min ||y − Xβ||² + α||β||₁
⚙️ PREPROCESSING
StandardScaler
كل الـ features اتعملها standardize (μ=0, σ=1) عشان الـ Ridge وـ Lasso يطبقوا الـ penalty بشكل عادل
Train/Test Split
80% Training (400 طالب) / 20% Testing (100 طالب) مع random_state=42 للـ reproducibility
Label Encoding
Parental_Involvement اتحولت من Low/Medium/High لـ 0/1/2 كـ ordinal variable
